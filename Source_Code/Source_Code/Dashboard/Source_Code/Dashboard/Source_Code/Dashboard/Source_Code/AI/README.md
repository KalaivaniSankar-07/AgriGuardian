# AgriGuardian AI Module

This module is planned to provide intelligent recommendations using field and environmental data.

## Planned Functions

- Irrigation recommendation
- Crop stress detection
- Soil-condition analysis
- Weather-based decision support
- Crop-yield prediction
- Fertilizer recommendation

## Current Status

The AI module is currently in the planning and development stage. Initial decision logic is implemented in the dashboard using sensor threshold values. Machine learning models will be added after collecting sufficient real-world field data.
